Lyapunov (soft) Actor-Twin Critic (LATC) algorithm.

This is a modified version of the Lyapunov Actor-Critic algorithm of Han et al. 2020. Like the original SAC algorithm, this LAC variant uses two critics instead of one to mitigate a possible underestimation bias, while the original LAC only uses one critic. For more information, see Haarnoja et al. 2018 or the LATC documentation.

Note

Code Conventions:

  • We use a _ suffix to distinguish the next state from the current state.

  • We use a targ suffix to distinguish actions/values coming from the target network.

Attention

To reduce the amount of code duplication, the code that implements the LATC algorithm is found in the LAC class. As a result this module wraps the lac() function so that it uses the LyapunovActorTwinCritic as the actor-critic architecture.
